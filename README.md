# 🛍️ Lojinha do Araújo

Loja de e-commerce simples, leve e sem dependências — HTML, CSS e JavaScript puro em um único arquivo.

## Como rodar

Abra o arquivo `index.html` no navegador. Não precisa instalar nada.

## Funcionalidades

- Catálogo de produtos com filtro por categoria
- Busca por nome em tempo real
- Carrinho lateral com ajuste de quantidade e total automático
- Finalização de compra com confirmação
- Layout responsivo (desktop e mobile)
